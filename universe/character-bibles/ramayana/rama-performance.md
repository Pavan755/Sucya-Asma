# Rama Performance Library v1

Purpose:
Define the emotional and behavioral language of Rama for Sucya-Asma productions.

Identity Version:
Rama_v1

---

# Performance State 1: Compassion

Core Feeling:
Understanding before judgment.

Visual Characteristics:
- Soft eyes.
- Relaxed facial muscles.
- Slight reassuring smile.
- Open posture.

Examples:
- Blessing the squirrel.
- Comforting others.
- Listening attentively.

Audience Impact:
Safety.
Warmth.
Trust.

---

# Performance State 2: Leadership

Core Feeling:
Calm guidance.

Visual Characteristics:
- Upright posture.
- Focused gaze.
- Controlled gestures.
- Confidence without intimidation.

Examples:
- Directing Vanaras.
- Strategic discussions.
- Leading by example.

Audience Impact:
Respect.
Confidence.
Security.

---

# Performance State 3: Reflection

Core Feeling:
Wisdom through contemplation.

Visual Characteristics:
- Gentle gaze into distance.
- Minimal movement.
- Thoughtful stillness.

Examples:
- Looking across the ocean.
- Quiet moments before decisions.

Audience Impact:
Curiosity.
Respect.
Peace.

---

# Performance State 4: Encouragement

Core Feeling:
Supporting others to become their best.

Visual Characteristics:
- Warm expression.
- Engaged attention.
- Gentle optimism.

Examples:
- Speaking to Lakshmana.
- Encouraging allies.

Audience Impact:
Hope.
Motivation.
Connection.

---

# Performance State 5: Protective Resolve

Core Feeling:
Strength in service of righteousness.

Visual Characteristics:
- Firm posture.
- Focused eyes.
- Controlled intensity.

Examples:
- Preparing for battle.
- Defending others.

Audience Impact:
Security.
Courage.
Determination.

---

# Performance State 6: Controlled Grief

Core Feeling:
Sorrow carried with dignity.

Visual Characteristics:
- Quiet sadness.
- Restrained expression.
- Composed demeanor.

Examples:
- Moments of separation.
- Reflection after loss.

Audience Impact:
Empathy.
Compassion.
Understanding.

---

# Performance Rules

Rama should never display:

- Mockery.
- Cruelty.
- Uncontrolled rage.
- Arrogance.
- Emotional instability.

---

# Usage Guidelines

Every Rama scene must reference:

Identity Version:
Rama_v1

AND

One approved Performance State.

Example:

Rama_v1
+
Compassion

or

Rama_v1
+
Leadership
